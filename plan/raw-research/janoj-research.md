# Janoj Research

## Slot Machine App Research

### Common Features Seen in Slot Machine Games
- Spin button
- Bet controls, usually either increase/decrease buttons or a place to enter a bet amount
- Win/loss message after each spin
- Sound effects for spinning and winning
- Payout table / paytable showing what symbol combinations are worth
- Bonus rounds or free spins
- Jackpot messaging
- Reset or restart button
- Extra visual feedback for bigger wins
- Auto-spin in some versions
- Game history or recent results in some versions

### Observations
- Important information is usually kept visible in an information bar the whole time, especially balance, current bet, and the most recent result.
- A lot of slot machine apps lean really hard into flashy visuals. That can make them look exciting, but it can also make the interface feel crowded.
- Some examples seem to care more about visual effects than making the controls easy to understand.
- For this project, it might be worth considering to go in a simpler direction so the game feels clearer and the code is easier to maintain.

## Slot Machine Terminology / Jargon

- **Reel**: the vertical column of symbols that spins
- **Payline**: the line across the reels used to determine winning combinations
- **Payout**: the amount awarded for a winning combination
- **Jackpot**: a very large payout, usually the highest possible reward
- **Symbol**: the icons/images shown on the reels
- **Wild**: a symbol that can substitute for others in some slot games
- **Scatter**: a symbol that may trigger bonuses regardless of payline position

### Notes
- Knowing these terms will probably help a lot when writing prompts
