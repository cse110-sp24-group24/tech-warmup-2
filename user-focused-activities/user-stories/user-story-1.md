# User Stories: Persona 1
## Responsive and Rewarding Feedback

**User Story:** As a gamer, I want the slot machine to give clear visual feedback when I win so that the machine feels entertaining.

**Acceptance Criteria:**
* The game displays a clear message after each spin indicating whether the player won or lost.
* Winning outcomes show a stronger visual feedback than losing outcomes.
* The game updates the result of a spin quickly enough so it doesn't feel delayed, but also delays it just long enough to give a sense of anticipation.
