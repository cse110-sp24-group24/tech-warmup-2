# User Stories: Persona 2
## Low Stakes

**User Story:** As a casual player, I want the ability to set my bet size to the absolute minimum (e.g., 1 token) so that I can stretch my gaming session out for as long as possible without running out of money.

**Acceptance Criteria:**
* The bet sizing tool allows a minimum input of 1.
* The machine clearly displays this as the lowest option.
* The "-" (decrease bet) button automatically disables or grays out when the bet amount is currently set to 1, preventing the user from accidentally betting 0 or negative tokens.
* When the user initially loads the game, the default bet amount is automatically set to 1 token to ensure they don't accidentally make a large bet on their very first spin.
