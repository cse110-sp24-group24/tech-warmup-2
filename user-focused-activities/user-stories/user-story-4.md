## User Stories: Persona 4

### Stats & Transparency Panel

**User Story:** As a curious/analytical user, I want to view statistics about my gameplay so that I can analyze patterns and understand how the system behaves over time.

**Acceptance Criteria:** 
- The interface displays basic stats (total spins, wins, losses, etc.)
- The stats update immediately after each spin.
- The stats panel is clearly visible but does not obstruct gameplay.
- The values shown are consistent with the actual outcomes of spins.
